# Torana logo

The official Torana mark — a stylized *torana* (gateway): a pointed finial,
layered eave beams, and two posts. Vector, `viewBox="1123 363 570 577"`.

| File | Use |
| --- | --- |
| `torana-color.svg` | Primary. Cyan→blue gradient (`#14F1D9`→`#0E8BE1`), for dark backgrounds. |
| `torana-white.svg` | Monochrome white, for photos / colored backgrounds. |
| `torana-dark.svg` | Monochrome black, for light backgrounds. |

All three share identical geometry — only the fill differs. The mark is a single
`fill-rule="evenodd"` path.

The Control Plane SPA header logo and SVG favicon
(`internal/controlplane/dist/index.html`) are inlined copies of the color variant.
